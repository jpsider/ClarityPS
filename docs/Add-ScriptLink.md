# Add-ScriptLink

## SYNOPSIS

## SYNTAX

```
Add-ScriptLink [[-Title] <String>] [[-SourceLink] <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### EXAMPLE 1
